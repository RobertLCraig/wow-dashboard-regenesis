You can interact with your team's data through API endpoints. By default, access to your team's data is disabled entirely. A private key will be generated when you enable access; with this key your entire environment is approachable, so keep it safe! To authenticate simply include the key as an Authorization header (prefixed with Bearer ) or as an `api_key` query parameter.
